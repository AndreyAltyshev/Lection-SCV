# Инструкция по работе с Markdown

## выделение текста

Чтобы выделить текст курсивом необходимо его обрамить звёздочкой (*). Например, *вот так*.

Чтобы выделить текст полужирным выделить его можно 2 звёздочками (**). Например **вот так**.
